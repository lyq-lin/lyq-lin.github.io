java学习之旅:关于jsp使用out.print方法错误问题

发表于 2021-03-02 更新于 2021-03-03 分类于 编程学习 阅读次数：
本文字数： 378 阅读时长 ≈ 1 分钟

前言

最近博主学习jsp过程中，遇到了out.print()使用报错问题，特此发布解决方法，以供有此问题朋友借鉴和以后再遇到此类问题不卡壳。

环境：

​ tomcat8.5.31

闲话少说，上图：

像上图一样，使用out.print()出现了

其实是tomcat的架包依赖没有导入 = =

打开idea选择file–>project Structure–>Dependencies，如图：

点击下方 + 号选择Library

选择相关依赖架包，这里博主使用的是Tomcat8.5.31

成功！

现在你可以用idea在jsp页面写java代码并且有智能提示~~

如果还报错建议重启一下idea 🤣感谢观看！~
